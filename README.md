# triangle-game
A game about triangles

Game Instructions:

Movement: WASD or Arrow keys
Shooting: Click Screen or "E" for autoshoot
Spin: Move mouse pointer of "C" for autospin
Weapons:
  1: Basic Tank
  2: Sniper
  3: Machine Gun
  4: Boomer
  5: Twin
  6: Shotgun
  F: Arena Closer ~(Not for normal gameplay)~
